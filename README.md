# my-bug-bounty-project
An advanced web vulnerability scanner to help improve security.

Features
Website Crawling:

Crawls all pages and links

Extracts forms and inspects parameters

Vulnerability Checks:

SQL Injection

Cross-Site Scripting (XSS)

Open Redirects

File Inclusion

Form Testing:

Tests all form fields

Validates POST and GET requests

Reporting:

Generates detailed JSON reports

Visualizes results

Stores reports in SQLite database

How to Use
python web_vulnerability_scanner.py example.com --depth 3
